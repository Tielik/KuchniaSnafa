API Admin


API aplikacji Kuchnia Snafa znajduje się pod adresem http://6186az.pythonanywhere.com/api.

Za jego pomocą można uzyskać informacje o składnikach i przepisach.

Dane są serializowane w formacie JSON

Dostęp do API wymaga wygenerowania unikalnego klucza API pod adresem: /api/token

API w input dostosowuje sie od inputu użytkownika
- jesli input to cyfra to szuka po ID
- jesli input to tekst to szuka po nazwie

Metody POST i PUT wymagają przesłania dodatkowo odpowiedniego wypełnionego JSON

GET, PUT, DELETE potrzebują inputu

/api/przepisy - zwraca liste przepisow z bazy danych
- POST: dodawanie nowego przepisu
- Składniki mogą być cyfrowo lub po nazwie (Pamiętaj o wielkości liter) ID lub nazwy można zobaczyć przy pomocy /api/skladniki

/api/przepisy/input - zwraca przepis o danym ID
- PUT: zmiana danego składnika w ListaPrzepisow podobnie jak w post
- DELETE: usuniecie danego przepisu po nazwie lub ID

/api/skladniki - zwraca liste składnikow z bazy danych
- POST: dodawanie nowego składnika (kategorie są od 1 do 7)
/api/skladniki/input - zwraca składnik o danym ID
- PUT: zmiana danego składnika
- DELETE: usuniecie danego składnika

Przykłady:
/api/przepisy/1 z metodą DELETE usunie przepis o ID = 1
/api/przepisy/1 z metodą PUT zmienia przepis o ID = 1.
Do tego trzeba dać JSON z wypełnioną chociaż jedną kolumną z bazy danych {% endblock %}
